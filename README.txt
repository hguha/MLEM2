TO USE:

1. Make sure you have python3 installed on your computer.
2. Navigate to this LEM2 folder
4. Run the command: python3 main.py

NOTE: The school Linux computers should have all dependacies needed, but in case any
      are missing(pandas, numpy, functools, itertools, os), simply type: pip3 install [name of library]

NOTE: It will ask you to enter in an input file.
      Any test files you wish to try should be placed in this folder, or accessed via their relative path. (i.e. "tests/iris.txt")

NOTE: It will also ask you for the name of the file you'd like to output to. It will output in this folder.
